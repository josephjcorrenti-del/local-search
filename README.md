# local_search

Local-first search engine for files and saved web artifacts.

Goals:
- explicit behavior
- inspectable data
- deterministic indexing
- CLI-first workflows
- reusable search independent from AI

Planned capabilities:
- local file indexing
- web artifact indexing
- keyword search
- ranked search results
- future retrieval integration with ollama_workbench

Status:
- early v1 bootstrap
